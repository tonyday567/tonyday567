## circuits

Free traced monoidal categories. Feedback as a first-class construction.

- [circuits](https://github.com/tonyday567/circuits) — the core library. `Circuit` GADT + `Hyper` final encoding + `Trace` class.
- [circuits-parser](https://github.com/tonyday567/circuits-parser) — parser combinators on `Circuit (->) Either`.
- [circuits-io](https://github.com/tonyday567/circuits-io) — bidirectional IO loops on `Circuit (Kleisli IO) Either`.
- [circuits-meter](https://github.com/tonyday567/circuits-meter) — performance measurement via `Circuit (Kleisli IO) (,)`.
